# AI Learning System Documentation

## Overview
This project is a self-contained AI learning system that supports local LLMs. It includes a chatbot, memory management system, and feedback processing.

## Features
- Local LLM engine using transformers/llama.cpp
- SQLite memory management
- Feedback processing for learning and improvement

## Installation
1. Clone the repository.
2. Install dependencies from `requirements.txt`.

## Usage
Run the chatbot with the command `python src/chatbot.py`.
